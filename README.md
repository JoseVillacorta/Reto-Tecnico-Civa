# Reto Tecnico Civa - Desarrollador Full Stack


Este proyecto es una solución full-stack desarrollada como parte de un reto técnico para la empresa **CIVA**. Consiste en un sistema de gestión de buses que permite administrar y visualizar información detallada de manera segura y eficiente.

## Descripción del Proyecto

El sistema se compone de una API robusta construida con **Spring Boot 3** y una interfaz de usuario dinámica desarrollada en **React 18**.
